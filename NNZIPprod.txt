### 3. `README.md` (replaced / enhanced)

```markdown
# nn-checkpoint

**ZIP-based neural network checkpoints** • Keeps only the best N • Framework-agnostic • Tiny & blazing fast.

```bash
pip install nn-checkpoint
```

### Features

- Single `.zip` file with architecture, weights (custom efficient binary), optimizer, metadata & full training log  
- `CheckpointManager` auto-keeps top-N checkpoints by any metric  
- CLI: `nn-inspect checkpoint.zip`  
- One-line PyTorch integration

### Install

```bash
pip install nn-checkpoint
# or from source
pip install -e .
```

### Quick Start

```python
from nn_checkpoint import Checkpoint, CheckpointManager

manager = CheckpointManager("checkpoints/", keep_top_n=3)

ckpt = Checkpoint(architecture, state_dict, optimizer_state=opt_state)
ckpt.log_epoch(...)

manager.save(ckpt, epoch=42)          # only saves if better
best = manager.best()                 # loads the best
```

See `examples/train_loop.py`

### CLI

```bash
nn-inspect checkpoints/epoch_0042.zip
```
